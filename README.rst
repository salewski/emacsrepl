About
-----

This is madness.  Less madness than readline though...

Usage
-----

- ``git clone https://github.com/tetracat/emacsrepl``
- ``cd emacsrepl``
- ``./emacsrepl``

Features
--------

- Basic readline-style editing (duh)
- Runs properly under ``M-x run-lisp``
- Persistent history
